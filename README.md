### About me

I'm a long-time Laravel enthusiast — solving challenging problems with the framework since **version 4**, and lucky to have shared space and codebases with many of the people who've shaped the community. I care about software that's a pleasure to maintain and holds up under real-world load.

Right now I'm focused on AI-augmented engineering: using tools like Claude and GitHub Copilot to move faster — sharper code reviews, broader test coverage, and quicker prototyping of architectural ideas. They don't replace the work; they amplify it.

- 🛠️ I like problems worth solving and codebases worth caring about.
- 🤝 I've spent a lot of my career mentoring, reviewing, and raising the bar for the teams around me.
- 📚 Always learning in public — you'll find write-ups on my [blog](https://anthonyterrell.com/blog).
- ♥️ I'm happiest when code turns into real-world positive impact.
